Descripción detallada

template<class T>
class VD< T >

T.D.A. VD.

Una instancia c del tipo de datos abstracto VD es un array 1d que almacena objetos de un tipo base. Esta especialmente diseñado para hacer accesos a la informacion de forma rapida. El conjunto de datos seria (a0, a1, ..., an).
